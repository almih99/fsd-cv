# double-date-dropdown

## Назначение

Блок представляет собой элемент интерфейса, отображающий рядом два блока date-dropdown.

## Структура

Структура результирующего элемента:

    .double-date-dropdown
      .double-date-dropdown__from-field
        +date-dropdown
      .double-date-dropdown__to-field
        +date-dropdown

## Использование

### Применение шаблона
    +date-dropdown(name, val1, val2, dddAttrs)

### Аргументы шаблона

Шаблон принимает аркументами:

1. Префикс имени элемента `<input type="hidden">`, используемого для передачи данных в форму. Полное имя формируется добавлением `"[0]"` и `"[1]"` к префиксу
2. Начальное содержимое первой метки.
2. Начальное содержимое второй метки.
3. Атрибуты охватывающего элемента.

### Значения

В качестве значения в форму передаются значения `<input type="hidden">` из вложенных календарей, атрибут `name` которого заполняется из параметра миксина с добавлением `"[0]"` и `"[1]"` для первого и второго календарей соответственно.

## Зависимости

Данный элемент зависит от элементa _date-dropdown_
